# 📄 AI-Based Resume Skill Extraction & Job Matching System

An AI-powered system that analyzes resumes and matches them with job descriptions using Natural Language Processing (NLP) and machine learning. This project helps automate resume screening and provides insights into skill gaps and job suitability.

---

## 🚀 Features

- 🔍 Resume and job description analysis  
- 🧠 NLP-based text preprocessing and skill extraction  
- 📊 TF-IDF feature extraction  
- 🤖 Multiple ML models (Logistic Regression, Naive Bayes, SVM, Random Forest)  
- 🏆 Best model selection based on performance metrics  
- 📈 Match score using cosine similarity  
- ✅ Matching and missing skill detection  
- 🌐 Interactive Streamlit web interface  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- NLP (custom preprocessing + keyword extraction)  
- Streamlit  

---

## 📊 Workflow

1. Dataset loading and exploration  
2. Text preprocessing  
3. Feature extraction (TF-IDF)  
4. Training multiple classification models  
5. Model evaluation and comparison  
6. Best model selection  
7. Resume–job matching and GUI implementation

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/RoAr29/AI-Based-Resume-Skill-Extraction-Job-Matching-System.git
cd AI-Based-Resume-Skill-Extraction-Job-Matching-System

```
2. Install dependencies:
```
pip install streamlit pandas numpy scikit-learn

```
3. Run the Streamlit app:
```
streamlit run app.py

```
4. Open in browser:
```
http://localhost:8501

```
